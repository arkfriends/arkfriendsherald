The Herald

A web-based, news content manager for electronic signage.

  • Administration
  • Content Manager
  • News Scheduler
  • Viewing Streamer

Key Components of this Service

The server has the following 4 components as well as a database to support these functions:

The Administration unit:
contains the kill switch, user account manager, emergency message broadcaster, and the service mode scheduler
The Content Manager:
allows new content to be uploaded and added to the system along with its metadata. There is support for the following file formats: Powerpoint PPTX, HTML, RSS, PDF, JPG, GIF, PNG, and TXT. Each news item is assessed a viewing priority to determine the frequency of its display. The range of dates for viewing are also recorded.
The News Scheduler:
creates a hotlist of active news items based on the relative priority of the the individual news items among the current content. The hotlist is used to synchronize the viewing of content between browsers.
The Viewing Streamer:
delivers the hot list items as html content that references the appropriate plugin to view the content

Requirements of the Electronic Signage Driver

The signage only requires a copy of the Chrome web browser installed with the following plugins:

Power Point Viewer:
capable of displaying and auto advancing the slides of files generated by Power Point (version 2013 or greater)
Adobe PDF Reader:
installed to auto scroll a PDF file within the browser window.
