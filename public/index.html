Components of the System

This system comprizes of 3 components and a databse to support these functions:

The Administration unit:
contains the kill switch, emergency message, the services scheduler
The Viewing Streamer:
provides the html content in measured type units
The Content Manager:
provides the means to add new content to the system

The Herald is a collaborative service at Payap University between the Office of Information Technology and Educational Innovation and the Office of University Communications
